# Współpraca / Contributing

Dziękujemy za zainteresowanie projektem! 🎉

## Jak możesz pomóc?

### 🐛 Zgłaszanie błędów
1. Sprawdź czy [Issue już nie istnieje](https://github.com/derekAllegro/kody-kreskowe-biedronka/issues)
2. Otwórz nowy Issue z opisem problemu
3. Dołącz kroki do reprodukcji błędu

### 💡 Propozycje funkcji
- Otwórz Issue z tagiem `enhancement`
- Opisz jaką funkcję chciałbyś zobaczyć i dlaczego

### 🔧 Pull Requesty
1. Forkuj repozytorium
2. Stwórz branch: `git checkout -b moja-zmiana`
3. Wprowadź zmiany
4. Wyślij PR z opisem co zmieniłeś

### 📝 Przykłady użycia API
Korzystasz z naszego API w swoim projekcie? Dodaj przykład do folderu `examples/`!

Akceptujemy przykłady w dowolnym języku programowania.

### 🌍 Tłumaczenia
README i dokumentacja API jest po polsku. Chętnie przyjmiemy tłumaczenia na inne języki.

## Zasady

- Bądź miły i szanuj innych 🤝
- Pisz czytelny kod z komentarzami
- Testuj swoje zmiany przed wysłaniem PR

## Pytania?

Otwórz Issue lub skontaktuj się przez formularz na [kodyzbiedry.pl](https://kodyzbiedry.pl).
